# DIY Gameboy Box Collection

Printable A4 box templates for various Game Boy games.

## Printing settings

- Format: A4
- Scale: 100%
- Paper: Glossy 135gsm recommended for the cover of the games
- Paper: standard 80gsm for the dimension paper
